ไม่ต้องกังวลครับ! ในบทความนี้ เราจะมาเริ่มต้นจากศูนย์กันจริงๆ ผมจะพาคุณสร้าง Workflow แรกที่ง่ายที่สุด เพื่อให้คุณเข้าใจหัวใจการทำงานของ n8n นั่นคือการสร้างข้อความ "สวัสดีชาวโลก" ขึ้นมาอัตโนมัติ หลังจากจบบทความนี้ คุณจะมีพื้นฐานที่แข็งแรงพอที่จะต่อยอดไปสร้างระบบที่ซับซ้อนขึ้นได้อย่างแน่นอน

สิ่งที่ต้องเตรียม: คอมพิวเตอร์และอินเทอร์เน็ต แค่นั้นเลยครับ!

ขั้นตอนที่ 1: เริ่มต้นกับ n8n Cloud

สำหรับมือใหม่ วิธีที่ง่ายที่สุดในการเริ่มต้นคือการใช้ n8n Cloud เพราะไม่ต้องติดตั้งโปรแกรมใดๆ ทั้งสิ้น แค่สมัครแล้วเริ่มใช้งานผ่านเว็บเบราว์เซอร์ได้เลย

  1. ไปที่เว็บไซต์ https://n8n.io/
  2. คลิกที่ปุ่ม "Start for free" หรือ "Sign up"
  3. สมัครสมาชิกด้วยวิธีที่สะดวกที่สุด (แนะนำให้ใช้บัญชี Google)
  4. เมื่อสมัครเสร็จเรียบร้อย คุณจะเข้ามาสู่หน้าจอหลัก (Dashboard) ซึ่งจะหน้าตาประมาณนี้

ขั้นตอนที่ 2: สร้าง Workflow แรกของคุณ

เมื่อเข้ามาที่หน้า Dashboard แล้ว ให้เราคลิกที่ปุ่ม "Start from scratch" เพื่อเริ่มสร้าง Workflow ใหม่จากกระดานเปล่า

2.1 เลือก "ตัวจุดประกาย" (Trigger)

หลังจากกด "Start from scratch" แล้ว n8n จะถามเราเป็นอย่างแรกว่า "จะให้ Workflow นี้เริ่มทำงานเมื่อไหร่?" หรือที่เรียกว่า Trigger นั่นเอง

สำหรับบททดสอบแรกของเรา ให้เลือกอันที่ง่ายที่สุดคือ Trigger manually ซึ่งหมายถึง "เริ่มทำงานเมื่อเรากดปุ่มเอง"

เมื่อเลือกแล้ว คุณจะได้ Node แรกหน้าตาแบบนี้มาวางอยู่บนกระดานทำงานของเรา

2.2 เพิ่ม "ตัวทำงาน" ชิ้นแรก (Set Node)

ต่อไปเราจะเพิ่ม "ตัวทำงาน" หรือ Node ชิ้นที่สองเข้ามา ให้คลิกที่เครื่องหมายบวก + ที่ต่อท้าย Node แรก

จากนั้น ในช่องค้นหาที่ปรากฏขึ้นมาทางขวา ให้พิมพ์คำว่า Set แล้วคลิกเลือก Edit Fields (Set)

Set Node คืออะไร? มันคือ Node พื้นฐานที่สุด มีหน้าที่ง่ายๆ คือ "สร้างหรือกำหนดค่าข้อมูล" เหมาะมากสำหรับการทดลองของเราครับ

2.3 ตั้งค่าให้ Set Node สร้างข้อความ

ตอนนี้เรามี Node สองชิ้นเชื่อมต่อกันแล้ว ให้เราคลิกที่ Edit Fields (Set Node) เพื่อตั้งค่า หน้าต่างตั้งค่าจะเปิดขึ้นมา ให้ทำตามนี้:

  1. คลิกที่ปุ่ม Add Field
  2. จะมี 2 ช่องปรากฏขึ้นมา:
    • ช่องบน (Name): พิมพ์ชื่อ myMessage (นี่คือชื่อของข้อมูลที่เราจะสร้าง)
    • ช่องล่าง (Value): พิมพ์ข้อความ สวัสดีชาวโลก (นี่คือค่าของข้อมูล)

ขั้นตอนที่ 3: ทดสอบและดูผลลัพธ์!

ถึงเวลาที่สนุกที่สุดแล้ว! หลังจากตั้งค่าเสร็จ เราจะมาทดสอบกันว่า Workflow ของเราทำงานได้จริงหรือไม่

ที่ด้านล่างของหน้าจอ ให้กดปุ่มใหญ่ๆ ที่เขียนว่า Execute workflow

รอสักครู่... คุณจะเห็นว่า Node ทั้งสองของเราจะมี กรอบสีเขียว สวยงามปรากฏขึ้นมาล้อมรอบ พร้อมกับข้อความ Workflow executed successfully ที่มุมขวาล่าง นี่คือสัญญาณว่า... สำเร็จแล้ว!

ขั้นตอนสุดท้าย: บันทึกผลงาน

อย่าลืมบันทึกผลงานชิ้นเอกของคุณไว้ด้วยล่ะ!

  1. ตั้งชื่อ: คลิกที่คำว่า My Workflow ด้านบนสุด แล้วเปลี่ยนเป็นชื่อที่จำง่าย เช่น บททดสอบแรกของฉัน
  2. บันทึก: กดปุ่ม Save สีส้ม/แดงที่อยู่ทางขวา
